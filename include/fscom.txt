C     Common contains the requested settings for settable parameters
C     and other useful information. 
C     Monitor-only parameters are also retained in common 
C     for the benefit of the monitor program. 
C     Whenever the monitor values are read from the device they 
C     are stored in common. 
C     The monitor-only quantities are marked with a  *  in the comments.
C     Settable parameters are indicated with a  +  below. 
C     Unmarked variables are not accessible directly via SNAP commands. 
C     Program TYCOM can be used to display the entire common block.
